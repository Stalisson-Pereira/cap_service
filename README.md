# Primeiros Passos

Bem-vindo ao meu novo projeto.

Ele contém as seguintes pastas e arquivos, seguindo o layout de projeto recomendado:

Arquivo ou Pasta | Finalidade
---------|----------
`app/` | conteúdo para interfaces de usuário (UI) fica aqui
`db/` | seus modelos de domínio e dados ficam aqui
`srv/` | seus modelos de serviços e código ficam aqui
`package.json` | metadados e configurações do projeto
`readme.md` | este guia de primeiros passos


## Próximos Passos

- Abra um novo terminal e execute `cds watch`
- (no VS Code, basta escolher _**Terminal** > Executar Tarefa > cds watch_)
- Comece a adicionar conteúdo, por exemplo, um arquivo [db/schema.cds](db/schema.cds).
